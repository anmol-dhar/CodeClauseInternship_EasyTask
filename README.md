# EasyTaskPro - Android Task Manager App

## Overview

EasyTaskPro is a powerful and user-friendly task manager app developed for Android devices. This app allows users to seamlessly manage their tasks by providing features such as task creation, editing, deletion, and secure cloud storage using Firebase. Users can sign up, log in, and access their tasks from any device, ensuring a seamless and consistent experience.

## Features

- **Task Management:** Easily add, edit, and delete tasks.
- **User Authentication:** Secure user authentication with Firebase authentication.
- **Cloud Storage:** All task data is stored securely in the cloud using Firebase Realtime Database.
- **Cross-Device Sync:** Access your tasks from any device by logging in with your account.
- **Logout Option:** Log out securely, knowing your tasks are safely stored in the cloud.

## Getting Started

### Prerequisites

- Android Studio: Make sure you have Android Studio installed on your development machine.
- Firebase Account: Create a Firebase account and set up a new project to enable cloud storage.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/anmol-dhar/CodeClauseInternship_EasyTaskPro.git

2. Open the project in Android Studio.

3. Connect the app to Firebase:

- Follow the Firebase setup guide to add your project to the Firebase console.

- Download the `google-services.json` file and place it in the `app` directory of your project.

4. Build and run the app on your Android device or emulator.

## Usage

1. Sign Up / Log In:

- Create a new account or log in with your existing credentials.

2. Task Management:

- Add a new task using the "Add Task" button.

- Edit or delete tasks as needed.

3. Logout:

- Log out securely using the "Logout" option.
